# README for Data Directory

Este diretório contém os dados utilizados no projeto de sistema solar. 

## Arquivos

- **sample_inputs.csv**: Este arquivo contém dados de entrada de exemplo que podem ser utilizados para testar o projeto. Os dados incluem informações sobre consumo de energia, localização e outros parâmetros relevantes para a simulação do sistema solar.

## Formato dos Dados

Os dados no arquivo `sample_inputs.csv` estão organizados em colunas, onde cada coluna representa uma variável específica necessária para a análise. Certifique-se de que os dados estejam no formato correto antes de utilizá-los no projeto.
